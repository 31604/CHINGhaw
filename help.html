Help & Documentation

How to use the Dashboard

  • Navigate through the menu to access Dashboard, Alerts, Devices, and Settings.
  • Dashboard shows live sensor readings and status.
  • Alerts page displays any triggered thresholds or notifications.
  • Devices page allows you to see all connected Raspberry Pi and sensors.
  • Settings page lets you configure Raspberry IP/Port and sensor thresholds.

Frequently Asked Questions (FAQ)

  • Q: How do I add a new sensor?
    A: The Raspberry Pi automatically detects new sensors connected and they appear in the Settings page.
  • Q: Can I access the dashboard remotely?
    A: Yes, by hosting the Raspberry Pi on a public IP or using port forwarding.
  • Q: How do I change thresholds for alerts?
    A: Go to Settings > Sensor Thresholds and set the values for each sensor.
