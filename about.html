About This Dashboard

Project Overview

This IoT Dashboard allows real-time monitoring of sensors connected to Raspberry Pi devices. You can view sensor data, set thresholds for alerts, and manage connected devices all from a single interface.

Developed By

Work Online Team

Version: 1.0.0

Technologies Used

  • HTML / CSS / JavaScript (Frontend)
  • Recharts for Graphs & Charts
  • Raspberry Pi & ESP Devices (IoT Backend)
  • MQTT or HTTP API Communication
